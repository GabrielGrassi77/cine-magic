# 🎬 CineMagic API Filmes

O **CineMagic API Filmes** é uma aplicação web que consome a API da **The Movie Database (TMDB)** para buscar e exibir informações de filmes em tempo real.

---

## 🔎 Sobre o projeto

Este projeto permite pesquisar filmes e visualizar detalhes como:
- Título do filme
- Sinopse
- Data de lançamento
- Idioma original
- Nota média
- Pôster oficial

---

## 🚀 Funcionalidades

- Busca de filmes por nome
- Exibição dinâmica dos resultados
- Tratamento para “nenhum resultado encontrado”
- Consumo de API externa (TMDB)
- Interface simples e funcional

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Fetch API
- API TMDB

---

## 🔑 Como usar a API

O projeto utiliza uma chave da API do TMDB.

No arquivo `script.js`, substitua:

```javascript
const apiKey = "SUA_API_KEY_AQUI"; 
```
Crie sua chave em
https://www.themoviedb.org/
